# 产品索引

## 项目是什么

当前项目是 `new-api` 代码库在本地 `AiLinkDog` 仓库中的承载版本。项目本体是一个 AI API 网关与代理系统，负责将多种上游大模型提供商统一封装为一致接口，并叠加用户、令牌、渠道、计费、限流、后台管理等能力。

从项目演进视角看，`AiLinkDog` 现在被定义为一个基于 `new-api` 的长期独立维护计划：

- 它是当前仓库未来持续优化与持续构建的起点
- 它允许本地路线逐步偏离上游实现细节
- 它要求长期跟踪上游能力变化，而不是完全脱离上游信息

## 核心能力

- 多上游模型提供商接入
- OpenAI / Claude / Gemini 等格式兼容与转换
- 用户与权限管理
- Token、渠道、模型策略管理
- 计费、比率、缓存计费、用量统计
- 管理后台与数据看板
- 多语言界面与文案
- OpenAI Responses / Realtime / Claude Messages / Gemini 等接口支持
- 智能路由、失败重试、用户级模型限流
- Thinking / reasoning 相关能力与格式转换

## 外部产品表达

## 上游与本地维护关系

- `new-api` 仍是当前仓库的来源项目与版权来源
- `AiLinkDog` 是当前本地长期维护路线的名称与内部认知入口
- 二者关系应被理解为“基于上游的长期独立维护计划”，而不是抹去上游来源
