# L2#Index

`L2#Index` 是索引层，用于把项目知识按主题组织成可导航的结构。

它不是详细规范正文，而是“去哪里找什么”的项目知识地图。

## 本层职责

- 给出产品、技术、结构、规范的索引入口
- 帮助未来 AI 快速定位 L3 规范
- 把根级 README、AGENTS、CLAUDE、代码结构整理成稳定导航

## 文件说明

- `product.md`：项目定位与核心能力
- `tech.md`：技术栈与工程约束
- `structure.md`：目录结构与职责分层
- `conventions.md`：开发约定与高优先级规则索引
- `toc.md`：L1/L2/L3 的统一导航目录

## 使用方式

建议先看 `toc.md`，再按主题进入对应文件。
