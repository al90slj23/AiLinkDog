# 元规范

## 目的

本目录用于保存当前项目的长期详细规范。

## 规则来源

当前规范来源按以下优先级确定：

1. 当前仓库代码
2. 当前 `AGENTS.md`
3. 当前 `CLAUDE.md`
4. 当前 `README*`
5. 旧项目 `.ai` 的层次结构灵感

## 明确约束

- 全程使用中文沟通交流
- specs 默认使用中文编写
- 旧项目中与当前仓库冲突的规则不得继承
- 当前代码与当前根级规范冲突时，以代码和现行高优先级规范为准
- `AiLinkDog` 可作为当前仓库长期维护计划名称存在，但不得用于删除或替换 `new-api` / `QuantumNous` 的受保护标识
