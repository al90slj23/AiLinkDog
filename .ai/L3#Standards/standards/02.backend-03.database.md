# 后端规范：数据库方向

## 适用范围

适用于数据库选型、文档说明、初始化流程和新增设计的默认数据库方向判断。

## 核心规则

- 从 `AiLinkDog` 开始，数据库方向正式收敛为 PostgreSQL-only。
- 新增设计与新规则不再为 SQLite/MySQL 编写兼容逻辑。
- PostgreSQL 是唯一正式数据库方向。
- 当前仓库中残留的 SQLite/MySQL 分支视为历史兼容代码，后续再系统清理。

## 注意事项

- 当前阶段优先收敛入口、文档、初始化与默认运行路径。
- 不再把 SQLite 或 MySQL 作为未来演进目标继续维护，也不再在文档或新入口中作为正式推荐数据库。
- 第一阶段仍优先使用 GORM 抽象，底层兼容代码的系统清理属于后续阶段。
