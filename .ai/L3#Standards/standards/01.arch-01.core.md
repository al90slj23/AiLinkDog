# 架构总览

## 后端主分层

当前项目以后端分层架构为核心：

`Router -> Controller -> Service -> Model`

## 关键辅助层

- `relay/`：上游协议适配与格式转换
- `setting/`：策略与配置
- `common/`：共享工具
- `dto/`：传输对象
- `types/`：共享类型

## 前端

前端是独立的 React + Vite + Semi Design 工程，位于 `web/`。

## 当前项目特殊点

这个仓库不是普通 CRUD 后台，而是 AI gateway，因此 `relay/` 与 `setting/` 的重要性非常高，任何架构理解都必须把这两层放进主视图中。
