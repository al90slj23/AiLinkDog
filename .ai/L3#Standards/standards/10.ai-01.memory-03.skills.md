# AI 记忆规范：技能

## 核心规则

- 本文件定义当前仓库中 AI skill 的项目内归档方式与执行工件定位。
- 项目内 skill 源文件统一归档在 `.ai/L0#Execution/skills/global/` 与 `.ai/L0#Execution/skills/project/`。
- `global/` 表示跨项目可复用的 skill 源文件，`project/` 表示只属于当前仓库的 skill 源文件。
- `L0` 是执行层，所以这里存放的是执行工件，而不是正式规范正文。

## 实施要求

- 新增 project skill 时，应优先放入 `.ai/L0#Execution/skills/project/`。
- 新增 global skill 时，应优先放入 `.ai/L0#Execution/skills/global/`。
- skill 是执行工件，正式规则必须写入 `L3`。
