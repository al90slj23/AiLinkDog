# L3 主规范目录

这里存放当前项目的主规范文件。

命名采用可无限扩展的单词链方式：

- `序号.单词-序号.单词.md`
- `序号.单词-序号.单词-序号.单词.md`

每一层只使用一个英文单词，单词必须在阅读全文后提炼得出。

当前按顶层主题分组：

- `00.*` 元规范
- `01.*` 架构规范
- `02.*` 后端规范
- `03.*` 前端规范
- `04.*` 质量规范
- `09.*` 工具规范
- `10.*` AI 记忆体系规范

维护约束：

- 命名以正文真实主主题为准，不直接沿用旧复合文件名
- 如果出现并列主题，优先拆文或增层，不使用 `and` 式复合命名
- 编号仅在直属父层内排序，不追求全局连续编号
- README 负责说明命名与分组，具体文件入口以 `../README.md` 与 `../../L2#Index/toc.md` 为准

所有规范均以当前仓库真实情况为准，不复制旧项目不适配的规则。
