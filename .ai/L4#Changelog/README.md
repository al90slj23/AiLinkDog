# L4#Changelog

`L4#Changelog` 是历史日志层，用于记录重要初始化、规范调整和关键决策。

## 本层职责

- 记录为什么修改
- 记录依据和取舍
- 让未来 AI 能追溯这套体系如何形成

## 当前内容

- 初始化日志：记录这次 `./ai` 六层记忆体系的建立过程

## 文件命名

- `L4` 日志文件默认使用“时间戳 + 主题”命名
- 推荐格式：`YYYY-MM-DD-HHMM-主题.md`
- 主题应直接表达本次工作的事实，不使用 `recent-*`、`latest-*` 这类滚动语义命名
- 一次独立工作或一次独立规则收敛，原则上对应一份独立日志文件
