# 术语知识补充

本文件用于沉淀当前项目中值得长期保持一致的关键术语。

主要来源：`docs/translation-glossary.md`

## 核心术语

- 倍率：Ratio
- 令牌：Token
- 渠道：Channel
- 分组：Group
- 额度：Quota

## 模型相关术语

- 提示：Prompt
- 补全：Completion
- 输入：Input / Prompt
- 输出：Output / Completion
- 模型倍率：Model Ratio
- 补全倍率：Completion Ratio
- 固定价格：Price per call
- 按量计费：Pay-as-you-go
- 按次计费：Pay-per-view

## 用户与充值相关术语

- 超级管理员：Root User
- 管理员：Admin User
- 普通用户：Normal User
- 充值：Top Up
- 兑换码：Redemption Code

## 渠道管理相关术语

- 密钥：Key
- 优先级：Priority
- 权重：Weight
- 代理：Proxy
- 模型重定向：Model Mapping

## 安全相关术语

- 两步验证：Two-Factor Authentication
- 2FA：Two-Factor Authentication

## 使用原则

- 多语言文案、文档、概览与规范中，尽量保持这些术语语义一致
- 当术语存在上下文歧义时，应优先保留当前项目的业务语义而不是逐词直译
- `Token` 在当前项目中可能同时表示 API 令牌、模型文本单元或访问令牌，必须结合上下文判断
