# 邀请返利文档索引

## 目的

本文件用于给当前项目中的邀请返利增强版相关文档建立一个快速索引，方便后续在以下三类问题之间快速切换：

1. 当前功能到底完成到了什么程度。
2. 这次数据库到底改了什么。
3. 上线前还需要怎样做高强度验证。
